Installation

ExcelPipe is simple to install - just double-click excelpipe.exe that you downloaded.

Note: You must have Microsoft Excel installed on your system.

The installation program creates uninstall entries under Control Panel:Add/Remove Programs. These allow ExcelPipe to be cleanly removed from your system.

Uninstall

Uninstallation is completely automatic.

To uninstall ExcelPipe, click the Start Menu, Settings, Control Panel, Add/Remove Programs. Select ExcelPipe from the list presented, and click Add/Remove. ExcelPipe will then be deleted.

The uninstall does not remove files modified or added by you, so any changes you have made to files will NOT be lost.
