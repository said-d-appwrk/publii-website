Files to process 

Include subfolders

When checked, the Include Subfolders box tells ExcelPipe to search all subfolders of the Start folder for spreadsheets to process. Clear this check box if you don't want to search subfolders of the Start folder.

When the file list is used, ExcelPipe will scan all subfolders for matching files.

The include subfolders box applies to all the types of file search.

Scan for files

Note: You can drag and drop files or folders to process directly from Windows Explorer to ExcelPipe and they will be added to the List of Files. If you drag and drop a settings file (.EPP) the new settings will be loaded.

Start folder

The start folder is where ExcelPipe starts searching for spreadsheets to process. You can browse to a new folder by clicking on the folder icon on the right hand side. To open the folder, double click the field.

To scan or make changes to a SharePoint site, first map your SharePoint site as a network drive.

File Types

The type of spreadsheets to find. You can separate multiple types with a semi-colon, e.g. *.xl*;*.ods

Start date

You can specify an optional start date/time, so that only files modified after this date will be examined.

End date

You can specify an optional end date/time, so that only files modified before this date will be examined.

Note: For Word documents (*.doc etc), use our WordPipe application. Note: For Excel documents (*.xls etc), use our ExcelPipe application. For Text documents (txt, html, xml, csv, tab, prn etc), use our TextPipe Pro application, for far greater speed.

Get filenames from a text file

Use this facility to load a list of files from a text file. Filenames should be one per line, and filespecs like c:\*.xls*;*.xlt can be used.

List of files

You can paste a list of filenames (and wildcards) directly into the multi-line edit box.

If your previous run had files with errors, you can copy the list of filenames from the Errors tab and paste the files into this box for reprocessing.

Import currently open files from Microsoft Excel

If you are working on files inside Microsoft Excel, you can easily transfer them to ExcelPipe with this button.

Add SharePoint URL

Click this button to add a URL pointing to a SharePoint site. If there is a SharePoint URL on the clipboard when you click this button, WordPipe will automatically try and browse to it.
