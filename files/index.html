Files

Open

Opens settings from a saved ExcelPipe settings file (.EPP). You can double-click .EPP files in Windows Explorer to open ExcelPipe with them.

Note - this is NOT used to open Microsoft Excel files. ExcelPipe is designed to process hundreds or thousands of Excel files, so it doesn't make sense to open them individually.

To specify which files to process, use the Files To Process tab.

Save

Saves the current settings to a ExcelPipe settings file.

Save As

Saves the current settings to a ExcelPipe settings file with a new name.

Recent Files

Contains a list of your most recently opened files.

Exit

Closes ExcelPipe.
