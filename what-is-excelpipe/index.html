What is ExcelPipe?

&

ExcelPipe allows you to quickly and easily search and replace across thousands of Excel spreadsheets.

ExcelPipe allows all user settings to be saved and opened so that the same tasks can be repeated in the future without re-keying.

It also has a search-only mode, so you can identify which documents contain text that needs to be manually fixed.
